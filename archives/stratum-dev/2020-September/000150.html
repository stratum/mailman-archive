There is also TRI_STATE_UNKNOWN for "don't set anything".

For debugging purposes it could help to check the built in ucli. Start
Stratum with the `-bf_switchd_background=false` flag.
You should then have a prompt to enter commands to the SDE:
bf-sde> pm show
-----+----+---+----+-------+----+---+---+---+--------+----------------+----------------+-
PORT |MAC |D_P|P/PT|SPEED  |FEC |RDY|ADM|OPR|LPBK    |FRAMES RX
|FRAMES TX       |E
-----+----+---+----+-------+----+---+---+---+--------+----------------+----------------+-
1/0  |23/0|128|3/ 0|40G    |NONE|YES|ENB|UP |  NONE  |               0|
          0|
2/0  |22/0|136|3/ 8|40G    |NONE|YES|ENB|UP |  NONE  |               0|
          0|

In the port manager you can try to get the port up with all kinds of
configuration options, without having to restart.
