Hi everyone,

Stratum's master branch is now *main*. You can update you local environment
with the following (or just re-clone the Stratum repo and main will be
checked out by default):

git branch -m master main
git fetch origin
git branch -u origin/main main

If you encounter any issues with the new branch name, please let us know.

Thanks, Rich and Max, for working through a lot of the code updates
required!
