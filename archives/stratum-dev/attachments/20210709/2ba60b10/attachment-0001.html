<div dir="ltr">We are pleased to announce the <span>Stratum</span> <b>2021-06-30</b> <span>release</span> (or <b>21.06</b> for short).<div><br></div><div>This release is chock full of features and fixes for Stratum's Intel Barefoot Tofino target, including support for two new SDE revisions, deflect on drop, and initial experimental support for QoS.</div><div><br></div><div>Stratum Tofino packages now include BSP support for the Wedge100BF, and users on those targets can select between the BSP or the JSON port mapping at startup. The JSON port mapping remains the standard approach that Stratum uses on all existing Tofino targets. The BSP provides better support for optical transceivers and supports additional hardware like gearboxes and retimers. We expect to increase the number of platforms that offer BSP support in future releases, and we encourage vendors to work with us to package their BSPs in Stratum.</div><div><br></div><div>In this release, Stratum now has support for finer grained control over logging via gNMI (see the release notes for more details). Stratum no longer listens for gRPC connections on port 28000 by default; users need to migrate to the registered IANA ports: 9339 for gNMI and 9559 for P4Runtime.<br><div><br></div><div><div dir="ltr"><div dir="ltr"><div><div>Pre-built Docker images and Debian packages, along with a more extensive changelist, are available in the GitHub <span>release</span> page:</div><div><a href="https://github.com/stratum/stratum/releases/tag/2021-06-30" target="_blank">https://github.com/stratum/stratum/releases/tag/2021-06-30</a></div><div></div></div><div><br></div><div>Enjoy!</div><div>Brian, Max, Yi and the rest of the <span>Stratum</span> team</div></div></div></div></div></div>
