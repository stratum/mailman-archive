<div><b>mkdir stratum-configs && cd stratum-configs</b></div>

</b></div>

</b></div>

<div><b>curl -o chassis_config.pb.txt <a href="https://raw.githubusercontent.com/stratum/stratum/master/stratum/hal/config/x86-64-stordis-bf2556x-1t-r0/chassis_config.pb.txt">
https://raw.githubusercontent.com/stratum/stratum/master/stratum/hal/config/x86-64-stordis-bf2556x-1t-r0/chassis_config.pb.txt</a></b><br>
</div>
<div><b>curl -o stratum.flags <a href="https://raw.githubusercontent.com/stratum/stratum/master/stratum/hal/bin/barefoot/deb/stratum.flags">
https://raw.githubusercontent.com/stratum/stratum/master/stratum/hal/bin/barefoot/deb/stratum.flags</a><br>
</b></div>
<div><b><br>
</b></div>
<div>4. Start Stratum with the script and some environment variable</div>
<div><b>PLATFORM=x86-64-stordis-bf2556x-1t-r0 DOCKER_IMAGE_TAG=9.1.0 CONFIG_DIR=$PWD LOG_DIR=$PWD ./start-stratum-container.sh</b><br>
</div>
<div><b><br>
</b></div>
