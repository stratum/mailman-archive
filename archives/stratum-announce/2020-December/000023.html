We are pleased to announce the Stratum *2020-12-18* release (or *20.12* for
short).

Pre-built Docker images and Debian packages, along with a more extensive
changelist, are available in the GitHub release page:
https://github.com/stratum/stratum/releases/tag/2020-12-18

This release adds a new, experimental *stratum_bfrt* target, and it
includes many bug fixes and usability enhancements.

Happy holidays, and thanks again to everyone that has participated in this
release!
Brian, Max, Yi and the rest of the Stratum team
